--- layout: post title: Set Up permalink: /docs/setup ---
  1. Download Atomic Docs and add to your local PHP environment. Rename "atomic-docs" to whatever the name of your project is.
  2. Open the config.php file in the root. Set what you would like the extension of your component files to be, E.G. .html, .php, .mustache (default is .html).
  3. Set up your preprocessor. The root preprocessed file must be named main.scss.
